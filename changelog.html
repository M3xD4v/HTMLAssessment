Version 3.13.2 - July 23, 2024
Added: Support for EPUB 3.3 fixed-layout documents, including improved typography and layout options.
Added: New annotation features, including sticky notes and highlighting.
Fixed: Issue with text reflow on certain PDF files, resulting in improved readability.
Improved: Performance of note synchronization, allowing for faster access to notes across devices.
Updated: User interface for better accessibility, including improved color contrast and font sizes.
Updated: Search functionality to include support for natural language queries.
